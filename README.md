# Data Cleaning Project

## Overview

This project focuses on cleaning and preprocessing the Airbnb NYC 2019 dataset. The goal is to improve data quality by identifying and handling common data issues such as missing values, duplicate records, inconsistent formatting, and outliers. A clean dataset helps ensure accurate analysis and reliable insights.

---

## Dataset Information

**Dataset Name:** [AB_NYC_2019-selected-columns.csv](https://www.kaggle.com/datasets/dgomonov/new-york-city-airbnb-open-data)

**Dataset Type:** Airbnb Listings Data

**Dataset Size:** 0.10 MB

The dataset contains information about Airbnb listings, including host details, neighbourhood groups, room types, prices, reviews, and availability.

---

## Objective

To improve the quality of the dataset by handling missing values, removing duplicate records, standardizing data formats, and detecting outliers, making the data ready for analysis and visualization.

---

## Challenges Performed

### 1. Data Integrity

Verified the accuracy, consistency, and reliability of the dataset by checking data types, missing values, and duplicate records.

### 2. Missing Data Handling

Identified missing values and handled them appropriately to improve data completeness and quality.

### 3. Duplicate Removal

Detected and removed duplicate records to maintain data uniqueness.

### 4. Standardization

Standardized data formats by ensuring consistent text formatting and removing unnecessary spaces.

### 5. Outlier Detection

Identified unusual values in numerical columns using the Interquartile Range (IQR) method.
